# 小鹿快传 - 在线P2P文件传输工具

### Get Started

1. Get code
```
git clone https://github.com/fanchangyong/deershare2
```

2. Install dependencies
```
cd server && yarn && cd ..
cd client && yarn && cd ..
```

3. Run server and client development server
```
cd server;
npm start
# 开另一个命令行窗口
cd client;
npm start
```
